git init .         -> Criar um repo git na pasta atual 
git status         -> Mostra o status do trabalho 
git add nome_arquivo -> Adicionar arquivo para ser comitado
git commit -m "msg" -> Faz commit do que estava para ser commitado 
git log -> Mostra todo o hostorico de commit

## Fluxo de trabalho

Antes de começar
Configurar nome e email para assinar commits:
git config - -global user.name “Seu nome”

git config - -global user.email “seu@email”



1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add nome_arquivo
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3.
